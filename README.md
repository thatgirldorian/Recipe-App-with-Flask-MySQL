# Recipe-App-with-Flask-MySQL

<h3>Function & Description:</h3>
This is a minimal recipe app that lets you:

- register a new user
- log in with the user created
- create a new recipe
- show the newly-created reipe on the dashboard


<h3>Technology Used:</h3>

- HTML
- CSS
- Python
- Flask
- MySQL
